# User Flow Requirements 
1. User can invite group members based on username or email or link
2. Users can join group through code 

3. Groups can create a head to head with members. 
4. Head-To-Heads enable the user to decide punishments and rewards for a binary decision in the future. 
5. The verification of this decision is made my head-to-head members votes.
